steps to run the camunda application:

1) download the sourse code from repository 
		link:- https://github.com/SJ00495527/Demo-Project.git
		
2) build the code through maven by running command on project root directory
		mvn clean install
3) run the application from project target directory through command prompt
       java -jar camunda-demo-project-1.0.0.jar
       
Helping Links:

1) Cockpit: http://localhost:8080/camunda/app/cockpit/default
